this folder contains the code to connect with ktools, see http://www.oasislmf.org/the-toolkit/
see https://github.com/davidnbresch/climada

to run on a cluster (such as Euler at ETH) with tools installed, you need to set
export PATH=$PATH:/cluster/apps/climate/ktools/bin
